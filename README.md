# Oppdrag brand assets

Assets publics de marque pour Oppdrag.Tech.

Ce dépôt centralise les fichiers sources SVG utilisables pour les profils sociaux, les bannières, les documents et les intégrations web. Les SVG sont les fichiers de référence ; certaines plateformes sociales demandent un export PNG aux dimensions recommandées.

## Fichiers principaux

| Fichier | Usage recommandé |
| --- | --- |
| `svg/01-icone-transparente.svg` | Favicon, avatar GitHub/Discord, watermark |
| `svg/02-logo-carre-dark.svg` | Photo de profil LinkedIn, X, Instagram, YouTube |
| `svg/03-logo-carre-light.svg` | Variante claire, documents, impression |
| `svg/04-logo-horizontal-dark.svg` | Site web, présentations, bandeaux sombres |
| `svg/05-logo-horizontal-light.svg` | Documents, PDF, pages claires |
| `svg/06-banniere-linkedin.svg` | Bannière LinkedIn personnelle, 1584 x 396 |
| `svg/07-banniere-reseaux-sociaux.svg` | Open Graph et partages sociaux, 1200 x 630 |
| `svg/08-entete-document.svg` | En-têtes de devis, contrats, propositions |
| `svg/09-signature-email.svg` | Signature email, export PNG conseillé |

## Palette

- Night : `#0A111F`
- Navy : `#121821`
- Ink : `#1B232F`
- Copper : `#C16A3E`
- Ivory : `#F2ECE0`
- Sand : `#CDB592`
- Cyan : `#36C5D0`
- Green : `#45B97C`

## Usage

Merci de conserver les proportions, les couleurs et les marges de sécurité des fichiers sources. Pour les usages publics, privilégier les exports haute résolution générés depuis ces SVG.
